JavaScript

explained

If HTML creates structure and CSS gives you a nice looking website, JavaScript makes it dynamic. It's a scripting language that lets you create content on a website that for example moves, refreshes or otherwise changes on your screen without you having to reload the page. Things the user might not even notice like when a website automatically updates or when you get suggestions typing in a question in a search bar. It is also used in game development and writing desktop applications.

Fun facts about JavaScript:

  • It is one of the most popular programming languages
  • It was designed in only 10 days about 25 years ago
  • Easy to learn as it runs on browser - just open your console and start coding
